1. Заполнение массива с консоли
2. Преобразование массива
3. Печать массива
4. Отбросить дробную часть у всех элементов массива

## 📋 Задания к лаб. 14

1.  Заполнение массива значениями из функции Func(x)
2.  Печать массива
3.  Вычисление суммы элементов массива
4.  Поиск в массиве значения равного a

## 📋 Задания к лаб. 16
1. Создать динамические массив, выполнить лаб. 11 зад. 2, сформировать новый массив
2. Заполнить массив случайными числами от -1 до 1
3. Выполните удаление всех элементов меньших среднего арифметического
4. Выполните вставку -999 перед первым отрицательным
5. Сгенерировать три массива a,b,c со случайными вещественными числами, количество элементов случайно в интервале от 10 до 50. Реализовать функцию, которая возвращает массив, составленный из последовательности массивов a,b,c с исключением минимальных значений в каждом из массивов


## 📋 Задания к лаб. 17
Реализовать функции сортировки массива целых чисел по следующему алгоритму:
1. Пузырьковой сортировки - void sort_bubble(int*ptrarr, int n);
2. Шейкерной (перемешиванием) сортировки - void sort_kokteil(int*ptrarr, int n);
3. Сортировки простым выбором - void sort_select(int*ptrarr, int n);
4. Сортировки вставками - void sort_insert(int*ptrarr, int n);

## 🔧 Реализованные функции

### 1. `double func(double x)`
- **Назначение**: Вычисление значения математической функции f(x) = |e^x - 2| + x^2
- **Параметры**: 
  - `x` - аргумент функции
- **Возвращает**: значение функции в точке x

### 2. `double* full_elements(double* ptr_array, int n)`
- **Назначение**: Заполнение массива значениями
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
- **Возвращает**: указатель на заполненный массив

### 3. `int put_elements(double* ptr_array, int n)`
- **Назначение**: Печать элементов массива
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
- **Возвращает**: 0 в случае успешной обработки

### 4. `double calc_elements(double* ptr_array, int n)`
- **Назначение**: Отбросить дробную часть у всех элементов массива
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
- **Возвращает**: 0 в случае успешной обработки, -1 при ошибке

### 5. `double sum_elements(double* ptr_array, int begin, int end)`
- **Назначение**: Вычисление суммы элементов массива в диапазоне [begin, end]
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `begin` - начальный индекс (включительно)
  - `end` - конечный индекс (включительно)
- **Возвращает**: сумму элементов

### 6. `int find_element(double* ptr_array, int n, double value)`
- **Назначение**: Поиск элемента в массиве
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
  - `value` - искомый элемент
- **Возвращает**: индекс найденного элемента или -1 если не найден

### 7. `int find_max_below_A(double* ptr_array, int n, double A)`
- **Назначение**: Поиск порядкового номера максимального элемента, который меньше заданного значения A
- **Параметры**: 
  - `ptr_array` - указатель на массив
  - `n` - размер массива
  - `A` - заданное значение
- **Возвращает**: индекс найденного элемента или -1 если нет элементов меньше A

### 8. `double* fill_random(double* ptr_array, int n, double min, double max)`
- **Назначение**: Заполнение массива случайными числами в заданном диапазоне
- **Параметры**: 
  - `ptr_array` - указатель на массив для заполнения
  - `n` - размер массива
  - `min` - минимальное значение
  - `max` - максимальное значение
- **Возвращает**: указатель на заполненный массив

### 9. `int remove_below_average(double* ptr_arr, int size)`
- **Назначение**: Удаление элементов меньших среднего арифметического
- **Параметры**: 
  - `ptr_arr` - указатель на массив
  - `size` - размер массива
- **Возвращает**: новый размер массива после удаления

### 10. `double* insert_before_first_negative(double* ptr_arr, int* size)`
- **Назначение**: Вставка числа -999 перед первым отрицательным элементом массива
- **Параметры**: 
  - `ptr_arr` - указатель на массив
  - `size` - указатель на размер массива
- **Возвращает**: указатель на измененный массив

### 11. `double find_min_value(const double* arr, int size)`
- **Назначение**: Поиск минимального элемента в массиве
- **Параметры**: 
  - `arr` - массив
  - `size` - размер массива
- **Возвращает**: минимальное значение

### 12. `int count_elements_excluding(const double* arr, int size, double exclude_value)`
- **Назначение**: Подсчет элементов в массиве, исключая заданное значение
- **Параметры**: 
  - `arr` - массив
  - `size` - размер массива
  - `exclude_value` - значение для исключения
- **Возвращает**: количество элементов, не равных exclude_value

### 13. `int copy_excluding_value(const double* src, int src_size, double* dest, int dest_index, double exclude_value)`
- **Назначение**: Копирование элементов из исходного массива в целевой, исключая заданное значение
- **Параметры**: 
  - `src` - исходный массив
  - `src_size` - размер исходного массива
  - `dest` - целевой массив
  - `dest_index` - текущий индекс в целевом массиве
  - `exclude_value` - значение для исключения
- **Возвращает**: новый индекс в целевом массиве

### 14. `double* arrays_D_without_min(const double* a, int size_a, const double* b, int size_b, const double* c, int size_c, int* result_size)`
- **Назначение**: Объединение массивов A, B, C в массив D с исключением минимальных значений
- **Параметры**: 
  - `a` - массив A
  - `size_a` - размер массива A
  - `b` - массив B
  - `size_b` - размер массива B
  - `c` - массив C
  - `size_c` - размер массива C
  - `result_size` - указатель на размер массива D
- **Возвращает**: указатель на полученный массив D

### 15. `void sort_bubble(double* ptarr, int n)`
- **Назначение**: Сортировка массива методом пузырька по убыванию
- **Параметры**: 
  - `ptarr` - указатель на массив для сортировки
  - `n` - размер массива

### 16. `void sort_kokteil(double* ptarr, int n)`
- **Назначение**: Сортировка массива методом шейкера по убыванию
- **Параметры**: 
  - `ptarr` - указатель на массив для сортировки
  - `n` - размер массива

### 17. `void sort_select(double* ptarr, int n)`
- **Назначение**: Сортировка массива методом выбора по убыванию
- **Параметры**: 
  - `ptarr` - указатель на массив для сортировки
  - `n` - размер массива

### 18. `void sort_insert(double* ptarr, int n)`
- **Назначение**: Сортировка массива методом вставок по убыванию
- **Параметры**: 
  - `ptarr` - указатель на массив для сортировки
  - `n` - размер массива
